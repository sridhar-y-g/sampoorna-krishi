# **App Name**: Sampoorna Krishi

## Core Features:

- Farmer-Centric Dashboard: Personalized dashboard displaying a farm overview and quick links to major tools.
- AI Market Price Forecasting: Provide AI-driven future price predictions for crops to inform selling decisions, acting as a decision-making tool.
- AI Weather Advisory: Deliver localized, crop-specific weather advisories using AI to help protect harvests, acting as a planning tool.
- Government Scheme Hub: Centralized hub to discover and apply for relevant government schemes.
- AI Crop Health Diagnosis: Diagnose crop health issues from uploaded photos using AI, providing treatment recommendations, acting as a decision-making tool.
- AI Subsidy Finder: Find applicable agricultural subsidies using natural language processing, acting as an informational tool.
- User Profile & Settings: Manage account details and app preferences, including language (English/Kannada).

## Style Guidelines:

- Primary color: Saturated teal (#008080), to evoke feelings of growth, health and freshness.
- Background color: Light grayish-teal (#E0F8F8), of similar hue as primary color but highly desaturated.
- Accent color: Analogous lime green (#B0F2B4) to complement the teal.
- Font pairing: 'Playfair' (serif) for headlines and 'PT Sans' (sans-serif) for body text. Playfair offers an elegant feel, while PT Sans ensures readability.
- Use modern, line-based icons to represent various agricultural elements and functionalities.
- Implement a clean, grid-based layout with ample whitespace for a user-friendly experience.
- Use subtle animations and transitions to enhance user engagement and provide feedback.
